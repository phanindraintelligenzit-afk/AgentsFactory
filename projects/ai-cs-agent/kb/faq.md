# AgentsFactory FAQ

## What is AgentsFactory?
AgentsFactory is a marketplace platform where businesses discover, deploy, and manage AI-powered agent teams for their workflows. Think of it as the "App Store for AI Agents."

## How do I get started?
1. Create a free account at https://aidentify.ai
2. Browse the marketplace for pre-built AI agent templates
3. Click "Deploy" on any agent to activate it in your workspace
4. Configure the agent with your preferences and connect your tools

## What is an AI Agent?
An AI Agent is an autonomous software program that uses large language models to perform tasks on your behalf. Examples include email triage, lead generation, content creation, and data analysis.

## How does billing work?
AgentsFactory uses a pay-as-you-go model:
- Free tier: 1,000 agent runs/month
- Pro plan ($29/mo): 10,000 agent runs/month
- Enterprise: Custom limits, dedicated support

## Can I build my own agents?
Yes! Use our Agent Builder to:
- Define agent capabilities and prompts
- Connect tools and APIs (Slack, Email, CRM, etc.)
- Test in sandbox mode before deploying
- Publish to the marketplace for others to use

## What integrations do you support?
We currently support:
- **Communication**: Slack, Email (Gmail, Outlook), Discord
- **Productivity**: Google Workspace, Notion, Airtable
- **Sales/CRM**: HubSpot, Salesforce, Pipedrive
- **Code**: GitHub, GitLab
- **Storage**: Google Drive, Dropbox, S3
- **Custom**: Use our REST API connector for any HTTP API

## How do I reset my password?
1. Go to https://aidentify.ai/login
2. Click "Forgot Password"
3. Enter your email address
4. Check your inbox for the reset link (arrives within 1-2 minutes)
5. Follow the link to set a new password

## What if I'm locked out of my account?
- Verify you're using the correct email address
- Check your spam folder for the reset email
- If you used SSO (Google/Microsoft), try signing in with that method instead
- Contact support@aidentify.ai if the issue persists

## How do I cancel my subscription?
1. Go to Settings → Billing
2. Click "Manage Subscription"
3. Scroll down and select "Cancel Plan"
4. You'll retain access until the end of your current billing period
5. Downgrade to Free tier takes effect immediately after cancellation

## Is my data secure?
Yes. AgentsFactory is SOC2 Type II certified. We use:
- AES-256 encryption at rest
- TLS 1.3 for data in transit
- Zero-trust architecture
- Regular third-party penetration testing
- Data never used for training models without explicit consent

## What is the SLA?
- Free tier: No SLA, best-effort support (48h response)
- Pro plan: 99.5% uptime, 24h email response
- Enterprise: 99.9% uptime, 1-hour emergency response, dedicated CSM

## Can I export my data?
Yes, all data can be exported:
- Go to Settings → Data Export
- Choose format (JSON, CSV)
- Full export delivered via secure download link within 24 hours
- GDPR/CCPA compliant data deletion available on request
