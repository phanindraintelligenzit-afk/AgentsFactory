# AI SOC2 Compliance Agent — Frontend

Built with React + Vite. Connects to the backend API at `http://localhost:8000`.

## Setup

```bash
npm install
npm run dev
```

Runs at `http://localhost:5173`

## Build

```bash
npm run build
```

## Features Dashboard

- SOC2 compliance overview with control completion status
- Evidence collection progress per control
- Audit timeline and milestone tracking
- Policy management with version control
- Integration status monitoring
